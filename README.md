# Segunda Evaluación - Minería de Datos

Este repositorio contiene el desarrollo de la segunda evaluación de la asignatura de Minería de Datos.

El objetivo del proyecto es predecir la variable `salary` utilizando modelos de machine learning aplicados a un dataset de salarios laborales.

## Modelos utilizados

- Regresión Lineal Múltiple
- Árbol de Decisión para Regresión

## Descripción

Primero se utiliza un modelo de Regresión Lineal Múltiple para estimar salarios a partir de variables como experiencia, educación, certificaciones, cargo, industria, ubicación, modalidad de trabajo y tamaño de empresa.

Luego se aplica un Árbol de Decisión para Regresión, comparando su desempeño con el modelo lineal mediante métricas como MAE, RMSE y R².

## Archivos

- `SegundaEvaluacionEC.ipynb`: notebook principal del proyecto.
- `job_salary.csv`: dataset utilizado.
